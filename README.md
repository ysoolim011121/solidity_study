WatsonNFT.sol 파일과, deploy.ts, testMint.ts을 업로드했다.  
WatsonNFT.sol 파일은 ERC-721 표준을 준수하는 NFT 스마트 컨트랙트이다.  
deploy.ts 파일은 Hardhat 환경에서 스마트 컨트랙트를 네트워크에 배포하는 스크립트이다.  
testMint.ts 파일은 배포된 컨트랙트가 정상 작동하는지 검증하는 시나리오 테스트 스크립트이다.  
단순히 Solidity 문법을 익히는 것을 넘어, Hardhat 개발 환경을 구축하고 테스트 코드를 작성하며 안정적인 DApp을 만드는 프로세스를 경험했다. mapping으로 데이터 조회 비용을 최적화하고, Event를 통해 중요한 데이터를 기록하는 패턴을 익혔다.  
로컬 환경에서의 검증을 마치고, Sepolia 테스트넷 배포를 준비할 예정이다.
